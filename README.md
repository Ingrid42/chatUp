# ChatUp

## Client

```bash
$ cd client
$ npm install #installer les dépendances
$ gulp #compiler le projet
$ node express.js #lancer le serveur web
```

## Serveur
```bash
$ cd serveur
$ ./compile.sh
$ ./run.sh
```
